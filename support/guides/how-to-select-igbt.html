选择合适的 IGBT 型号是设计高效功率转换系统的关键。本文将从电压等级、电流等级、开关频率、封装形式等方面详细介绍 IGBT 选型方法。

一、确定电压等级

首先根据应用的母线电压确定 IGBT 的耐压等级。一般规则是：

  • 400V 系统 → 选择 600V/650V IGBT
  • 690V 系统 → 选择 1200V IGBT
  • 1140V 系统 → 选择 1700V IGBT

考虑电压裕量，通常选择额定电压为最大工作电压的 1.5-2 倍。

二、确定电流等级

根据负载电流和过载要求确定 IGBT 的电流等级。需要考虑：

  • 连续工作电流
  • 峰值电流
  • 散热条件

一般建议选择额定电流为最大工作电流的 1.5-2 倍。

三、考虑开关频率

不同应用对开关频率的要求不同：

  • 工业变频：2-20kHz
  • 伺服驱动：10-50kHz
  • 太阳能逆变：10-100kHz

高频应用应选择开关损耗低的型号，如 IGBT7 系列。

四、选择封装形式

根据功率等级和散热要求选择封装：

  • 小功率（<10kW）：TO-247、TO-220 等分立封装
  • 中功率（10-100kW）：EconoDUAL™ 3、EasyPACK™ 等模块
  • 大功率（>100kW）：PrimePACK™、IHV 等大功率模块

五、其他考虑因素

  • 是否需要短路保护功能
  • 工作温度范围要求
  • 是否需要集成续流二极管
  • 成本预算

六、选型实例

应用：50kW 工业变频器，690V 电网

选型过程：

  1. 电压等级：690V × 1.5 = 1035V → 选择 1200V IGBT
  2. 电流等级：50kW / (√3 × 690V × 0.85) ≈ 49A，考虑 2 倍裕量 → 选择 100A 以上
  3. 开关频率：10kHz → 选择标准 IGBT4 或 IGBT7
  4. 封装：50kW 功率等级 → EconoDUAL™ 3 封装

推荐型号：

FF100R12ME4 或 FF150R12ME4
